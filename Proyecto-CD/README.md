# Proyecto-CD

## Clone the project

- git clone https://github.com/CristhianMotoche/Proyecto-CD.git
- cd Proyecto-CD

## Create Virtual Env
- virtualenv-3.4 .

## Active virtual environment
- source bin/activate

## Install dependencies
- pip3 install -r requirements.txt

## Run Harmonic
- cd Harmonic 
- python3 main.py

## Run Huffman
- cd Huffman
- python3 main.py
*Go to the begining of the output.*
